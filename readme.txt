== Description ==

This plugin was created to for those people who are unsatisfied with image resizing capabilites of wordpress uploader. 
It uses services from flickr. You can upload your images to flickr.com and then make your photos public. You can play with your pictures using services provided by flickr and then using this plugin you can easily attach them to your blog post.

This release aadded the option for choosing tags or sets to fetch the images. Also fixed the issue where enabling this plugin, interfered with wordpress 2.5's jquery.
you can get more information at [akWpUploader article](http://amiworks.co.in/talk/akwpupoader-alternative-wordpress-image-uploader/ " Alternative Wordpress Image Uploader using Flickr").


After you check out this plugin, please provide some rating and leave comments for improving it.

PLEASE NOTE ! This plugin is still in development and may contain few bugs, please report any bug that you find !,

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `akwpuploader-alternative-wordpress-image-uploader` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it nothing else.
1. Now you can use it when you create new posts.
== Frequently Asked Questions ==

= How to use it? =

simple just type your flickr ID in the text area and it will fetch the images from flickr for you. Once images are their just click on the image that you want to attach to you post as you do normally.
